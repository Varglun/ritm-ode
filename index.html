You can use this app to solve numerically first-order ODE of type: \(y' = f(t, y)\).

Attention! Use accurately as there is no check for the bad initial conditions.

Math function Designation
\(a^b\) Math.pow(a,b)
\(e^t\) Math.exp(t)
\(\ln{t}\) Math.log(t)
\(\sin{t}\) Math.sin(t)
\(\cos{t}\) Math.cos(t)
\(\tan{t}\) Math.tan(t)
\(\arcsin{t}\) Math.asin(t)
\(\arccos{t}\) Math.acos(t)
\(\arctan{t}\) Math.atan(t)
\(\pi\) Math.PI
